## Ghost API errors

| Error | Cause | Fix |
| --- | --- | --- |
| `401 Unauthorized` | Invalid or expired Admin API key | Regenerate and update your Ghost credentials |
| `422 Unprocessable Entity` | Invalid member data | Check the email format in the event log |
| `404 Not Found` | Wrong Ghost Admin URL | Verify the URL in your credentials settings |

## Webhook errors

| Error | Cause | Fix |
| --- | --- | --- |
| Webhook not received | Wrong URL in provider | Copy the correct URL from your mapping |
| Signature mismatch | Wrong webhook secret | Re-copy the webhook secret from PayGlue |

## Still stuck?

Contact [team@payglue.io](mailto:team@payglue.io) and include the event ID from your webhook event log.
