## What you need

- Your Ghost Admin URL (e.g. `https://yoursite.ghost.io`)
- A Ghost Admin API key with Staff permissions

## Steps

<Steps>
  <Step title="Open Ghost Admin">
    Go to your Ghost dashboard and navigate to **Settings > Integrations**.
  </Step>
  <Step title="Create a custom integration">
    Click **Add custom integration**, give it a name (e.g. PayGlue), and copy the **Admin API key**.

    <Note>Placeholder: Ghost Admin — Settings > Integrations, custom integration with API key visible</Note>
  </Step>
  <Step title="Add to PayGlue">
    In your PayGlue dashboard, go to **Connection > Ghost CMS** and paste the Admin URL and API key.

    <Note>Placeholder: PayGlue Dashboard — Connection > Ghost CMS form</Note>
  </Step>
</Steps>
