Polar is the first fully supported payment provider in PayGlue.

## Prerequisites

- A Polar account at [polar.sh](https://polar.sh)
- At least one product created in Polar

## Steps

<Steps>
  <Step title="Create an API token">
    In Polar, go to **Settings > Preferences > Developers** and create a new API token. Set expiration to **Never** and enable the required scopes.

    <Note>Placeholder: Polar — Settings > Preferences > Developers, new API token form</Note>
  </Step>
  <Step title="Add to PayGlue">
    In your PayGlue dashboard, go to **Connection > Polar** and paste your API token.

    <Note>Placeholder: PayGlue Dashboard — Connection > Polar, token field</Note>
  </Step>
  <Step title="Copy the webhook URL">
    On the same connection page, copy the **Webhook URL** and add it in **Polar > Settings > Webhooks**.

    <Note>Placeholder: PayGlue Dashboard — Connection > Polar, webhook URL section</Note>
  </Step>
  <Step title="Create a mapping">
    Open the **Buy Button**, **Paywall**, or **Pricing Table** editor and select your Polar product. The mapping is created automatically when you save.
  </Step>
</Steps>

<Note>
  PayGlue uses Polar's `order.paid` event to trigger Ghost membership creation.
</Note>
